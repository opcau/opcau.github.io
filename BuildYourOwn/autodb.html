Autonomous Data Warehouse Cookbook

Learn how to set up your Autonomous Data Warehouse, configure and test it

Pre-requisites:

1.  SQL Developer version 18.1 or later (available here: http://www.oracle.com/technetwork/developer-tools/sql-developer/overview/index.html)

2.  A functioning Oracle Cloud Infrastructure account or trial account (preferably in one of the US regions)

Instructions:

Open the Autonomous Data Warehouse console in the main (left-hand) menu:

 

2.  Select the “root” compartment if not already set:

 

 3.  Click “Create Autonomous Data Warehouse”

 

 

4.  Enter the following details in the “Create Autonomous Data Warehouse” dialog:


Display Name: ADWTest
Database Name: ADWTest
CPU Core Count: 1
Storage (TB): 1

Administrator Password: A complex password of your choosing
Confirm Password: As above

Select “Subscribe to New Database Software Licenses and the Database Cloud Service”
