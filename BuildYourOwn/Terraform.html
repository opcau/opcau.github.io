Learn how to Install Terraform onto your windows PC

Download Terraform locally

1.       Browse to https://www.terraform.io/downloads.htm

2.       Scroll down to locate the Windows 64-bit download.

3.       Locate the download and run on your local machine.

4.       Create a folder on your /my documents/ called ‘Tools’

5.       Copy the ‘terraform.exe’ into the ‘Tools” folder

 

Create a local path to call Terraform.exe when you run in DOS

6.       Browse to Control Panel > System > Advanced System Settings > Environment Variables

7.       In the ‘system Variables box click on ‘path’ and edit

8.       Select ‘New’ and type the full path to your ‘Tools folder you created.  In my case it was C:\Users\%username%\Documents\Tools

9.       To test.  Open up a command window.  Enter ‘terraform version’.
